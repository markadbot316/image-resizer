# image-resizer

Browser image resizer with single-image and bulk ZIP export flows.

## Caveman integration

This project now includes a repo-local Caveman setup based on
`JuliusBrussee/caveman`.

- `.codex/hooks.json` auto-activates Caveman-style brevity when Codex starts in this repo.
- `.agents/skills/caveman/SKILL.md` adds a local Caveman skill definition for future sessions.

To stop it in-chat, use `normal mode` or `stop caveman`.
